üöÄ Main Deployment Pipeline

startUploadS3.ps1 - Complete Deployment

# Navigate to deployment folder cd startUpload # Full deployment with validation .\startUploadS3.ps1 -Message "Your deployment message" # Skip validation (faster) .\startUploadS3.ps1 -SkipValidation -Message "Quick deployment" # Force deployment even if validation fails .\startUploadS3.ps1 -Force -Message "Emergency deployment"

What the pipeline does:

  1. Validation: Checks prerequisites and environment
  2. S3 Deployment: Builds React app and deploys to S3
  3. Content-Type Fix: Ensures proper MIME types
  4. Git Backup: Commits and pushes to GitHub
‚úÖ Pipeline Benefits:
  • One command deploys everything
  • Automatic content-type fixes
  • Git backup after successful deployment
  • Comprehensive error handling

‚ö° git-upload.ps1 - Git Operations

Usage from git folder:

# Navigate to git folder cd git # Initial upload (first time) .\git-upload.ps1 -Initial # Regular updates with custom message .\git-upload.ps1 -Message "Fixed mobile responsiveness issues" # Quick update (auto message) .\git-upload.ps1

Smart Features:

  • Automatically initializes Git repository if needed
  • Adds GitHub remote using your GitHub username
  • Handles merge conflicts automatically
  • Skips upload if no changes detected
  • Shows progress with colored output

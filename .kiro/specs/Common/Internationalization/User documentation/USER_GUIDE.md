# myAdmin Language Guide

Welcome to myAdmin! This guide will help you use myAdmin in your preferred language.

## Supported Languages

myAdmin is available in:

- **🇳🇱 Nederlands (Dutch)** - Primary language
- **🇬🇧 English** - Secondary language

All features, menus, buttons, and messages are fully translated in both languages.

---

## Changing Your Language

### Quick Steps

1. **Navigate to the Dashboard** (if you're on another page)
2. **Find the language selector** in the top-right corner of the dashboard header (next to your profile)
3. **Click the language button** showing your current language (e.g., "🇳🇱 Nederlands")
4. **Select your preferred language** from the dropdown menu
5. The page will refresh and display in your chosen language

### Where to Find the Language Selector

The language selector appears in the **main dashboard header**:

```
┌─────────────────────────────────────────────────────┐
│ myAdmin Dashboard          🇳🇱 Nederlands ▼  [User] │
└─────────────────────────────────────────────────────┘
```

Click on "🇳🇱 Nederlands" (or "🇬🇧 English") to open the language menu.

**Note**: If you're on a different page (Reports, Banking, STR, Admin), navigate back to the Dashboard to access the language selector.

---

## Your Language Preference is Saved

Once you select a language, myAdmin remembers your choice:

✅ **Across all pages** - Your language stays the same as you navigate
✅ **After logging out** - Your preference is saved when you log back in
✅ **On all devices** - Use the same language on your computer, tablet, or phone
✅ **In emails** - System emails (like invitations) are sent in your preferred language

## What Changes When You Switch Languages

### User Interface

All text in the application changes to your selected language:

- **Navigation menus** - Dashboard, Reports, Banking, STR, Admin
- **Buttons** - Save, Cancel, Delete, Export, etc.
- **Form labels** - All input fields and dropdowns
- **Messages** - Success messages, error messages, warnings
- **Help text** - Tooltips and instructions

### Dates and Numbers

Formatting automatically adapts to your language:

**Dutch Format:**

- Dates: 18-02-2026 (day-month-year)
- Numbers: 1.234,56 (period for thousands, comma for decimals)
- Currency: €1.234,56

**English Format:**

- Dates: 2/18/2026 (month/day/year)
- Numbers: 1,234.56 (comma for thousands, period for decimals)
- Currency: €1,234.56

### Reports and Exports

- **Report titles** and column headers appear in your language
- **Excel exports** include translated column headers
- **Charts and graphs** show labels in your language
- **PDF reports** display in your language

### Email Notifications

System emails are sent in your preferred language:

- **User invitation emails** - Welcome message and instructions
- **Password reset emails** - Security instructions
- **Notification emails** - Alerts and updates

## Language-Specific Features

### Dutch (Nederlands)

- **Primary language** for Dutch businesses
- **Tax forms** use Dutch terminology (BTW, IB, Toeristenbelasting)
- **Date format** follows Dutch standard (DD-MM-YYYY)
- **Number format** uses Dutch conventions (1.234,56)

### English

- **International language** for global users
- **Tax forms** use English terminology (VAT, Income Tax, Tourist Tax)
- **Date format** follows US standard (MM/DD/YYYY)
- **Number format** uses US conventions (1,234.56)

## Frequently Asked Questions

### How do I change my language?

Click the language selector in the top-right corner of any page and choose your preferred language.

### Will my language choice affect other users?

No, each user has their own language preference. Your choice only affects what you see.

### Can I use different languages on different devices?

Your language preference is saved to your account, so you'll see the same language on all devices where you're logged in.

### What if I accidentally select the wrong language?

Simply click the language selector again and choose the correct language. The page will refresh immediately.

### Are all features available in both languages?

Yes! Every feature, button, menu, and message is fully translated in both Dutch and English.

### Can I switch languages in the middle of working?

Yes, you can switch languages at any time. Your work is saved, and the page will refresh in the new language.

### What language will I see when I first log in?

New users see the default language set by your organization (usually Dutch). You can change it immediately using the language selector.

### Are reports and exports translated?

Yes, all reports, charts, and Excel exports appear in your selected language with properly formatted dates and numbers.

### What about email notifications?

System emails are sent in your preferred language. If you haven't set a preference yet, emails use your organization's default language.

### Can I request a new language?

Currently, myAdmin supports Dutch and English. Contact your system administrator if you need additional languages.

## Tips for Using myAdmin in Your Language

### 1. Set Your Language First

When you first log in, take a moment to set your preferred language. This ensures the best experience from the start.

### 2. Check Date Formats

Pay attention to date formats when entering data:

- Dutch: DD-MM-YYYY (18-02-2026)
- English: MM/DD/YYYY (2/18/2026)

### 3. Number Entry

When entering numbers, use the format for your language:

- Dutch: Use comma for decimals (1234,56)
- English: Use period for decimals (1234.56)

### 4. Search and Filters

Search terms work in both languages. For example, searching for "invoice" or "factuur" will find the same records.

### 5. Help and Support

If you need help, contact support in your preferred language. Our team is fluent in both Dutch and English.

## Troubleshooting

### Language selector not visible

**Solution**: The language selector only appears on the main Dashboard page. If you're on another page (Reports, Banking, STR, Admin), click "Dashboard" in the navigation menu to return to the main page where the language selector is located.

### Page not translating after language change

**Solution**: The page should refresh automatically. If it doesn't, manually refresh your browser (F5 or Ctrl+R).

### Wrong date or number format

**Solution**: Check that you've selected the correct language. The format automatically matches your language choice.

### Emails in wrong language

**Solution**: Your language preference may not be saved. Change your language using the selector, then check your profile settings.

### Some text still in wrong language

**Solution**: This is rare. Try clearing your browser cache and logging in again. If the problem persists, contact support.

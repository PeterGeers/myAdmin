# Revenue Summaries

> View revenue, occupancy, and trends for your short-term rentals.

## Overview

myAdmin offers comprehensive reporting for your STR activities. You can view revenue per listing, channel, and period, and analyze trends with interactive charts.

## Available reports

STR reports are available via **Reports** → **BNB** tabs:

| Report                | What it shows                                                  |
| --------------------- | -------------------------------------------------------------- |
| **BNB Revenue**       | Revenue per listing/channel, broken down by year/quarter/month |
| **BNB Actuals**       | Year-over-year net revenue totals                              |
| **Violin charts**     | Distribution of price per night and nights per stay            |
| **Returning guests**  | Guests who booked more than once                               |
| **Future revenue**    | Planned bookings aggregated per date/channel/listing           |
| **Tourist tax**       | Calculated tourist tax per period                              |
| **Country of origin** | Guest origin analysis by country                               |

## BNB Revenue

The main report shows revenue grouped by:

- **Listing** — Which rental property
- **Channel** — Airbnb, Booking.com, VRBO, or Direct
- **Period** — Year, quarter (Q1-Q4), or month (1-12)

**Displayed amounts:**

| Column      | Description                     |
| ----------- | ------------------------------- |
| Gross       | Total booking price             |
| Channel fee | Platform commission             |
| VAT         | Calculated VAT                  |
| Tourist tax | Calculated tourist tax          |
| Net         | Paid out to owner               |
| Nights      | Total number of overnight stays |
| Bookings    | Number of bookings              |

## Future revenue

This report shows planned bookings as a trend line:

- Aggregated per date, channel, and listing
- Shows expected revenue for the coming months
- Updated with each import of booking files

!!! info
Future revenue is based on planned bookings and may change due to cancellations or new reservations.

## Violin charts

Violin charts show the distribution of:

- **Price per night** — How your prices are distributed per listing/channel/year
- **Nights per stay** — How long guests stay on average

These charts help you recognize pricing patterns and seasonal effects.

## Returning guests

This report shows guests who booked more than once:

- Guest name
- Number of bookings
- Total revenue

Useful for identifying your loyal guests.

## Step by Step

### View a revenue report

1. Go to **Reports**
2. Click the **BNB** tab
3. Select the desired report (Revenue, Actuals, Violins, etc.)
4. Use the filters to select the period and listing

### Update future revenue

1. Go to **STR**
2. Import the latest booking files
3. Future revenue is automatically updated

## Tips

!!! tip
Compare year-over-year via the **BNB Actuals** report to see growth trends.

- Use violin charts to see if your pricing is consistent
- Check tourist tax per quarter for your declaration
- The "Country of origin" report helps understand your guest profile

## Troubleshooting

| Problem                 | Cause                | Solution                                                                   |
| ----------------------- | -------------------- | -------------------------------------------------------------------------- |
| No data in reports      | No bookings imported | Import booking files via the STR module first                              |
| Future revenue is empty | No planned bookings  | Import a file with future bookings                                         |
| Amounts don't match     | Wrong tax rates      | Check that check-in dates are correct — rates are determined automatically |

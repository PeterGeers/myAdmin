# Omzet overzichten

> Omzet, bezetting en trends bekijken voor je kortetermijnverhuur.

## Overzicht

myAdmin biedt uitgebreide rapportages voor je STR-activiteiten. Je kunt omzet bekijken per accommodatie, kanaal en periode, en trends analyseren met interactieve grafieken.

## Beschikbare rapporten

De STR-rapportages zijn beschikbaar via **Rapportages** → **BNB** tabbladen:

| Rapport                 | Wat het toont                                                        |
| ----------------------- | -------------------------------------------------------------------- |
| **BNB Omzet**           | Omzet per accommodatie/kanaal, uitgesplitst naar jaar/kwartaal/maand |
| **BNB Actuals**         | Jaar-op-jaar netto omzettotalen                                      |
| **Viooldiagrammen**     | Verdeling van prijs per nacht en nachten per verblijf                |
| **Terugkerende gasten** | Gasten die meer dan één keer hebben geboekt                          |
| **Toekomstige omzet**   | Geplande boekingen geaggregeerd per datum/kanaal/accommodatie        |
| **Toeristenbelasting**  | Berekende toeristenbelasting per periode                             |
| **Land van herkomst**   | Analyse van gastherkomst per land                                    |

## BNB Omzet

Het hoofdrapport toont de omzet gegroepeerd per:

- **Accommodatie** — Welk verhuurobject
- **Kanaal** — Airbnb, Booking.com, VRBO of Direct
- **Periode** — Jaar, kwartaal (Q1-Q4) of maand (1-12)

**Getoonde bedragen:**

| Kolom              | Beschrijving                 |
| ------------------ | ---------------------------- |
| Bruto              | Totale boekingsprijs         |
| Kanaalkosten       | Platformcommissie            |
| BTW                | Berekende BTW                |
| Toeristenbelasting | Berekende toeristenbelasting |
| Netto              | Uitbetaald aan eigenaar      |
| Nachten            | Totaal aantal overnachtingen |
| Boekingen          | Aantal boekingen             |

## Toekomstige omzet

Dit rapport toont geplande boekingen als trendlijn:

- Geaggregeerd per datum, kanaal en accommodatie
- Toont verwachte omzet voor de komende maanden
- Wordt bijgewerkt bij elke import van boekingsbestanden

!!! info
Toekomstige omzet is gebaseerd op geplande boekingen en kan wijzigen door annuleringen of nieuwe reserveringen.

## Viooldiagrammen

Viooldiagrammen tonen de verdeling van:

- **Prijs per nacht** — Hoe je prijzen verdeeld zijn per accommodatie/kanaal/jaar
- **Nachten per verblijf** — Hoe lang gasten gemiddeld blijven

Deze grafieken helpen je om prijspatronen en seizoenseffecten te herkennen.

## Terugkerende gasten

Dit rapport toont gasten die meer dan één keer hebben geboekt:

- Gastnaam
- Aantal boekingen
- Totale omzet

Handig om je trouwe gasten te identificeren.

## Stap voor stap

### Omzetrapport bekijken

1. Ga naar **Rapportages**
2. Klik op het **BNB** tabblad
3. Selecteer het gewenste rapport (Omzet, Actuals, Violen, etc.)
4. Gebruik de filters om de periode en accommodatie te selecteren

### Toekomstige omzet bijwerken

1. Ga naar **STR**
2. Importeer de nieuwste boekingsbestanden
3. De toekomstige omzet wordt automatisch bijgewerkt

## Tips

!!! tip
Vergelijk jaar-op-jaar via het **BNB Actuals** rapport om groeitrends te zien.

- Gebruik viooldiagrammen om te zien of je prijzen consistent zijn
- Controleer de toeristenbelasting per kwartaal voor je aangifte
- Het rapport "Land van herkomst" helpt bij het begrijpen van je gastenprofiel

## Problemen oplossen

| Probleem                  | Oorzaak                     | Oplossing                                                                         |
| ------------------------- | --------------------------- | --------------------------------------------------------------------------------- |
| Geen data in rapporten    | Geen boekingen geïmporteerd | Importeer eerst boekingsbestanden via de STR-module                               |
| Toekomstige omzet is leeg | Geen geplande boekingen     | Importeer een bestand met toekomstige boekingen                                   |
| Bedragen kloppen niet     | Verkeerde belastingtarieven | Controleer of de incheckdatums correct zijn — tarieven worden automatisch bepaald |

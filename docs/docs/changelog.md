# Changelog

> Overzicht van wijzigingen per versie.

## April 2026

### Nieuw

- **Documentatiesite** — Volledige gebruikersdocumentatie beschikbaar via `/docs/` en in-app hulp
- **Tweetalig** — Documentatie beschikbaar in Nederlands en Engels
- **In-app hulp** — Hulpicoon op elke pagina opent relevante documentatie

### Verbeteringen

- **E-mailsysteem** — Alle e-mails (uitnodigingen, wachtwoord-resets, tenant-toevoegingen) gaan nu via één systeem (SES) en worden bijgehouden in het e-maillogboek
- **Wachtwoord reset** — Reset-e-mails komen nu van `support@jabaki.nl` in plaats van `no-reply@verificationemail.com`
- **Gebruiker verwijderen** — Verbeterde beveiliging: gebruikers met toegang tot meerdere tenants worden niet meer per ongeluk volledig verwijderd
- **Uitnodiging opnieuw versturen** — Als de e-mail niet kan worden verstuurd, wordt het tijdelijke wachtwoord getoond
- Eerste release van de gebruikersdocumentatie

---

_Eerdere wijzigingen worden hier toegevoegd bij toekomstige releases._

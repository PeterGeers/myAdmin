# Welkom bij myAdmin

> De complete documentatie voor myAdmin financieel beheer.

## Wat is myAdmin?

myAdmin is een alles-in-één platform voor financieel beheer. Het helpt je bij het verwerken van banktransacties, facturen, kortetermijnverhuur (STR) en belastingaangiften — allemaal vanuit één plek.

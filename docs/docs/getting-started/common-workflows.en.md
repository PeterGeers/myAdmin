# Common Workflows

> The key tasks you'll perform regularly in myAdmin.

## Overview

This is a quick reference for the most common tasks. Each workflow links to detailed documentation for more information.

## Monthly tasks

### Import bank statements

1. Download your bank statement as CSV (Rabobank format)
2. Go to **Banking**
3. Click **Import** and select the file
4. Review the imported transactions
5. Click **Apply Patterns** to automatically assign accounts

→ More: [Importing statements](../banking/importing-statements.md)

### Process invoices

1. Go to **Invoices**
2. Click **Upload** and select one or more PDF files
3. Wait for the AI to extract the details
4. Review and correct the results
5. Click **Approve** to save the invoice

→ More: [Uploading invoices](../invoices/uploading-invoices.md)

### Import STR bookings

1. Download the revenue file from Airbnb or Booking.com
2. Go to **STR**
3. Click **Import** and select the file
4. View the realized and planned bookings

→ More: [Importing bookings](../str/importing-bookings.md)

## ZZP tasks

### Create and send an invoice

1. Go to **ZZP** → **Invoices**
2. Click **New invoice** and select the contact
3. Add line items from your product catalog
4. Review the totals and click **Save**
5. Open the draft and click **Send**

→ More: [Creating invoices](../zzp/creating-invoices.md) | [Sending invoices](../zzp/sending-invoices.md)

### Log time entries

1. Go to **ZZP** → **Time Tracking**
2. Click **New entry**
3. Select the contact, enter the hours and rate
4. Click **Save**
5. Create an invoice from the logged hours

→ More: [Time tracking](../zzp/time-tracking.md)

### Check payments

1. Import your bank statements via **Banking**
2. Go to **ZZP** → **Debtors**
3. Click **Check payments**
4. The system automatically matches payments with outstanding invoices

→ More: [Debtors & creditors](../zzp/debtors.md)

## Periodic tasks

### Prepare VAT declaration

1. Go to **Tax** → **BTW**
2. Select the period (quarter)
3. Review the calculated amounts
4. Export the overview

→ More: [VAT declaration](../tax/btw.md)

### View reports

1. Go to **Reports**
2. Choose the desired dashboard or report
3. Set the period
4. Export to Excel if needed

→ More: [Dashboards](../reports/dashboards.md)

### Check Google Drive links

1. Go to **PDF Validation**
2. Click **Start Validation**
3. Review the results and fix broken links

→ More: [Checking links](../pdf-validation/checking-fixing-links.md)

## Tips

!!! tip
Always start in **Test mode** when performing a new workflow for the first time. This way you can practice without risk.

- Most imports show a progress bar — wait until it completes
- Use **Apply Patterns** after every bank import to save time
- Always review AI extraction results for invoices before approving

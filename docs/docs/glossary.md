# Woordenlijst

> Uitleg van financiële en vakspecifieke termen die je tegenkomt in myAdmin.

## Nederlands → Engels

| Term                                | Engelse vertaling             | Uitleg                                                               |
| ----------------------------------- | ----------------------------- | -------------------------------------------------------------------- |
| **Aangifte**                        | Declaration / Tax return      | Een officiële opgave aan de Belastingdienst                          |
| **Aangifte IB**                     | Income tax return             | Jaarlijkse aangifte inkomstenbelasting                               |
| **Afschrift**                       | Bank statement                | Overzicht van transacties van je bankrekening                        |
| **Balans**                          | Balance sheet                 | Overzicht van bezittingen en schulden op een bepaald moment          |
| **Belastingdienst**                 | Tax authority                 | De Nederlandse overheidsinstantie voor belastingen                   |
| **BNB**                             | BNB (Bed and Breakfast)       | Kortetermijnverhuur accommodatie                                     |
| **Boekjaar**                        | Fiscal year                   | De periode waarover je administratie voert (meestal 1 jan - 31 dec)  |
| **BTW**                             | VAT (Value Added Tax)         | Belasting over de toegevoegde waarde, omzetbelasting                 |
| **Crediteur**                       | Creditor                      | Partij aan wie je geld verschuldigd bent (leverancier)               |
| **Crediteurenbeheer**               | Accounts payable              | Beheer van openstaande schulden aan leveranciers                     |
| **Creditnota**                      | Credit note                   | Document dat een eerder verstuurde factuur (gedeeltelijk) corrigeert |
| **Debiteur**                        | Debtor                        | Partij die jou geld verschuldigd is (klant)                          |
| **Debiteurenbeheer**                | Accounts receivable           | Beheer van openstaande vorderingen op klanten                        |
| **Duplicaat**                       | Duplicate                     | Een transactie die al eerder is geïmporteerd                         |
| **Factuur**                         | Invoice                       | Rekening voor geleverde goederen of diensten                         |
| **Grootboekrekening**               | General ledger account        | Rekening in de boekhouding voor het categoriseren van transacties    |
| **Huurinkomsten**                   | Rental income                 | Inkomsten uit verhuur van vastgoed                                   |
| **Inkomstenbelasting (IB)**         | Income tax                    | Belasting over je inkomen                                            |
| **Kwartaal**                        | Quarter                       | Periode van drie maanden (Q1-Q4)                                     |
| **KvK-nummer**                      | Chamber of Commerce number    | Uniek registratienummer bij de Kamer van Koophandel                  |
| **Mutatie**                         | Transaction / Mutation        | Een financiële boeking of wijziging                                  |
| **Mutaties**                        | Transactions                  | Meervoud van mutatie — alle financiële boekingen                     |
| **Omzet**                           | Revenue / Turnover            | Totale inkomsten uit bedrijfsactiviteiten                            |
| **Omzetbelasting**                  | Sales tax / VAT               | Synoniem voor BTW                                                    |
| **Patroon**                         | Pattern                       | Regel voor automatische toewijzing van rekeningen aan transacties    |
| **Periode**                         | Period                        | Tijdvak voor rapportage (maand, kwartaal, jaar)                      |
| **STR**                             | STR (Short-Term Rental)       | Kortetermijnverhuur via platforms als Airbnb                         |
| **Tenant**                          | Tenant                        | Een organisatie of gebruikersgroep binnen myAdmin                    |
| **Toeristenbelasting**              | Tourist tax                   | Gemeentelijke belasting op overnachtingen                            |
| **Urenregistratie**                 | Time tracking                 | Bijhouden van gewerkte uren per klant en project                     |
| **Verouderingsanalyse**             | Aging analysis                | Overzicht van openstaande facturen per ouderdomscategorie            |
| **Winst- en verliesrekening (W&V)** | Profit & Loss statement (P&L) | Overzicht van inkomsten en uitgaven over een periode                 |
| **ZZP**                             | Self-employed (no employees)  | Zelfstandige Zonder Personeel — freelancer                           |
